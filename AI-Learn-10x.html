📋 Prompt หลักสำหรับ NotebookLM (คัดลอกได้เลย) 
ฉันกำลังศึกษาหัวข้อ "[ใส่หัวข้อ เช่น การย้ายไปอยู่บัลแกเรีย]". 
ฉันมีพื้นฐานระดับ [ใส่ระดับตัวเอง เช่น มีพื้นฐานนิดหน่อย]. 
ช่วยทำ 4 อย่าง:
1. สรุปเฉพาะ insight ใหม่ ๆ ที่ฉันยังไม่รู้ (ตัดส่วนซ้ำซากทั้งหมด)
2. เขียนเป็น Markdown bullet points ชัดเจน
3. สร้างตารางเปรียบเทียบ (ถ้ามีหลายประเทศ/หลายทางเลือก)
4. ใส่แหล่งอ้างอิงทุกจุด + Audio Overview + Mind Map
ผลลัพธ์ที่ได้: Audio Overview (ฟังได้เลย), Mind Map, Flashcard, Test, Infographic, Markdown สรุปพร้อมแหล่งอ้างอิง

2 Gemini = นักคิด + เขียน + ตัดสิน (Synthesis & Personalize)

  1. 01

    เปิด Gemini (gemini.google.com) หรือ Gemini Advanced

    กด “Attach” → เลือก NotebookLM ที่เพิ่งทำเสร็จ (เชื่อมกันอัตโนมัติ)

  2. 02

    Paste Prompt นี้ (คัดลอกได้)

    นี่คือสรุปจาก NotebookLM: [วางทั้งหมด]
    ฉันรู้เรื่องนี้ระดับ [บอกระดับตัวเอง]

    ช่วยทำ 3 อย่าง:
    1. ตัดส่วนที่ฉันรู้แล้วออกหมด
    2. เขียนเป็น bullet points พร้อมแหล่งอ้างอิง
    3. สร้าง script / ตาราง / แนะนำส่วนที่ต้องลึกกว่านี้
  3. 03

    ตัวอย่างผลลัพธ์

    • • Script รีวิวหนังสือ 3 นาที (พร้อม hook + conclusion)
    • • ตารางเปรียบเทียบประเทศ (พร้อมคะแนนส่วนตัว)
    • • คำแนะนำ “ข้ามเล่มนี้ได้” + เหตุผล

3 Obsidian = สมองที่สองถาวร (Second Brain)

วิธีโยนข้อมูลเข้า Obsidian

1. Export จาก NotebookLM หรือ Gemini

Copy Markdown → วางใน note ใหม่

2. ใช้ [[Link]] เชื่อมกับ note เก่า

เช่น [[ความรู้การเงินของฉัน]]

3. สร้าง MOC (Map of Content)

หน้าเดียวสรุปทุกอย่างที่เชื่อมโยง

ทำไม Obsidian สำคัญสุด?

  • 🔗 ความรู้ใหม่เชื่อมกับความรู้เก่า → ไม่ลืม
  • 📈 กลายเป็นเครือข่ายความรู้ส่วนตัว
  • ⏳ ครั้งหน้าจะเรียกใช้ได้ทันที
